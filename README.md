## About
<img src="https://user-images.githubusercontent.com/81938036/156874202-80a8799d-df4d-40db-8257-6490294aa628.png" width="256" height="545"> <img src="https://user-images.githubusercontent.com/81938036/156874362-6a454d61-ec13-4994-9f54-7303bf8c5f57.png" width="256" height="545">

<img src="https://user-images.githubusercontent.com/81938036/156874396-5532f2cd-3cae-4ba0-aedb-86adac963eb6.png" width="256" height="545"> <img src="https://user-images.githubusercontent.com/81938036/156874401-d90924da-3f86-432b-900c-bab39a18ce9b.png" width="256" height="545">

Suppose you have classified music albums into each directory.  
Do you want to see countless album directories in your gallery?  
Maybe someday, but it won't be like that normally.  
How about replacing directories to be expressed by managing daily slots and album slots separately?  

The slot gallery is simple, but it presents a new paradigm for the gallery.
<br><br>

## Basic Features
Like other applications, Slot has the basic features that a gallery should have.
### Sort
<img src="https://user-images.githubusercontent.com/81938036/156874908-c3a1feb5-0703-47d7-ac8d-910f4ad87f9f.png" width="256" height="545"> <img src="https://user-images.githubusercontent.com/81938036/156874912-f21c5c55-1024-4d0c-ba96-dde316dd7b7a.png" width="256" height="545">

Contents can be sorted by Name, Newest, and Oldest.  
The selection of directory and content sorting is saved separately.
<br><br>

### Share
<img src="https://user-images.githubusercontent.com/81938036/156875198-853a082b-d9bf-44c3-86e4-70bed0c32b9a.png" width="256" height="545"> <img src="https://user-images.githubusercontent.com/81938036/156875199-1da8cae4-84bd-4bbd-ba03-11cfaa63cbf2.png" width="256" height="545">

Contents can be shared with Application Chooser.  
Sharing can be done individually in image activities as well as in multi-selects of directories or contents.
<br><br>

### Copy
<img src="https://user-images.githubusercontent.com/81938036/156875198-853a082b-d9bf-44c3-86e4-70bed0c32b9a.png" width="256" height="545"> <img src="https://user-images.githubusercontent.com/81938036/156875361-06c5cad0-8679-4311-9485-221146185c2d.png" width="256" height="545">

Contents can be copied with Choice Activity.  
Copy can be done individually in image activities as well as in multi-selects of directories or contents.
<br><br>

### Delete
<img src="https://user-images.githubusercontent.com/81938036/156875198-853a082b-d9bf-44c3-86e4-70bed0c32b9a.png" width="256" height="545"> <img src="https://user-images.githubusercontent.com/81938036/156875457-15b93f23-dd4f-4512-8f01-28e4321c1d6e.png" width="256" height="545">

Contents can be deleted with Media Store.  
Deletion can be done individually in image activities as well as in multi-selects of directories or contents.
<br><br>

### Rename
<img src="https://user-images.githubusercontent.com/81938036/156875554-2ca9dbd9-dc39-438f-bb89-2d54c19577d3.png" width="256" height="545">

Contents can be renamed with Media Store.  
Rename can be done only in image activities.  
Format changes are not allowed.
<br><br>

### Information
<img src="https://user-images.githubusercontent.com/81938036/156875656-e04d0c84-ffdf-4e79-b4fd-303496a21225.png" width="256" height="545"> <img src="https://user-images.githubusercontent.com/81938036/156875658-7c8a6e3d-2787-44aa-83ea-7d6dd6eea2f7.png" width="256" height="545">

Contents Information can be checked with Image Activity.
<br><br>

## Special Features
Unlike other applications, Slot Gallery provide special features with Settings.
### Manage Slot
<img src="https://user-images.githubusercontent.com/81938036/156875840-6d6be45e-1f13-4906-82e6-c9b6143c3558.png" width="256" height="545"> <img src="https://user-images.githubusercontent.com/81938036/156875842-42d2433b-d083-464d-a687-6bb9256ea3ae.png" width="256" height="545">

You can create and manage slots.  
Create slots for each theme and display only the content you want to show.
<br><br>

### Manage Directory
<img src="https://user-images.githubusercontent.com/81938036/156875840-6d6be45e-1f13-4906-82e6-c9b6143c3558.png" width="256" height="545"> <img src="https://user-images.githubusercontent.com/81938036/156875953-db779f0b-cae7-438b-b8bb-a50f5bfbffa7.png" width="256" height="545">

You can add and manage directories to selected slot with Document Tree.  
Only the content of the directory you added is displayed in the gallery.
